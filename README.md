## Description

Code Cultivation est mon deuxieme projet de la picine python de 42. Le but est de toujours ce famialiariser et de voir les base du python avec des exercices simples sur le theme du jardin.

## Detail

| Exercices | Role |
|-----------|------|
| ex0 | Le but de cet exercice est d'afficher simplement du texte et nous montrer comment faire un main et mettre la condition du if __name__. |
| ex1 | Le but de cet exercice est de voir les class simplement et comment les prints. |
| ex2 | Le but de cet exercice est de voir les class et comment on peu evoluer une class. |
| ex3 | Le but de cet exercice est de voir les class avec des list et une fonction show. |
| ex4 | Le but de cet exercice est de voir les class mais notamment comment les securises avec les set et get. |
| ex5 | Le but de cet exercice est de voir les class ainsi que les heritages avec la fonction super(). |
| ex6 | Le but de cet exercice est de voir la difference entre staticmethod et classmethod en re utilisant tout ce qu'on appris dans le module. |
